🚌 BusReserve - Advanced Bus Ticket Booking System
BusReserve is a full-stack Python application built with FastAPI that provides a seamless interface for searching, holding, and booking bus tickets in real-time.

🔗 Live Demo
Check out the live application here: https://bus-booking-system-r85w.onrender.com/

✨ Key Features
Real-time Seat Selection: Visual seat layout with "Temporary Hold" mechanism to prevent double-booking.

Advanced Search: Filter buses by origin, destination, date, and bus type (AC/Non-AC).

User Authentication: Secure login and registration with profile-linked QR codes.

Admin Dashboard: Dedicated interface for managing buses, routes, schedules, and viewing revenue stats.

Live Tracking: Simulation of real-time bus locations based on trip progress.

🛠️ Technical Stack
Backend: FastAPI (Python)

Database: SQLite

Templating: Jinja2

Scheduler: APScheduler (for handling expired seat holds)

Styling: HTML5, CSS3, and JavaScript
